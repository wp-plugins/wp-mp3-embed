== Description ==

Search and embed mp3 from mp3.zing.vn (top 1 website in Vietnam).

Feature:

- Search mp3 song info and insert to content
- Change title and size before insert

== Installation ==

1. Unzip the wp-mp3-embed.zip
2. Copy wp-mp3-embed folder to wp-content/plugins
3. Go to Plugins/Installed Plugins, find WP-MP3 Embed and click Active
4. Enjoy!
